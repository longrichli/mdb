# mdb客户端和mdb服务器通信协议

# 命令报文结构
```txt
+-----------------+---------------------+-----------------+
|    bytes        |     type            |    name         |
+-----------------+---------------------+-----------------+
|    2            |     u16             |    length       |
+-----------------+---------------------+-----------------+
|    length       |     u8 array        |    content      |
+-----------------+---------------------+------------------
```
# content结构

```txt
----------------------------------------------------------
commentStr'\r\n'keyStr'\r\n'valStr1'\r\n'valStr2'\r\n'valStrn'\r\n'
----------------------------------------------------------
```

# 回复报文结构
```txt
+-----------------+---------------------+-----------------+
|    bytes        |     type            |    name         |
+-----------------+---------------------+-----------------+
|    1            |     s8              |   response code |
+-----------------+---------------------+-----------------+
|    2            |     u16             |    length       |
+-----------------+---------------------+-----------------+
|    length       |     u8 array        |    content      |
+-----------------+---------------------+------------------
```

# response code 含义

```txt
+---------------+--------------+
| response code | describtion  |
+---------------+--------------+
| -1            | error        |
+---------------+--------------+
| 0             | ok           |
+---------------+--------------+
| 1             | string       |
+---------------+--------------+
| 2             | array        |
+---------------+--------------+
| 3             | nil          |
+---------------+--------------+
```
